 # 🧠 Memory Card Game

A fun and interactive **Memory Card Game** built using **HTML**, **CSS**, and **JavaScript**. Test your memory by flipping cards and matching pairs! Designed with an attractive UI, smooth animations, and responsive layout — perfect for learning JavaScript logic or adding to your portfolio.

## 🎮 How to Play

1. Click any two cards to flip them.
2. If they match, they stay flipped.
3. If not, they flip back after a short delay.
4. Try to match all the pairs with the fewest moves possible!

## ✨ Features

- 🃏 Flip animation and matching logic
- 🎨 Modern, responsive design
- 🔁 Restart functionality
- 🧠 Keeps track of flipped cards and matched pairs
- 📱 Fully mobile-friendly
